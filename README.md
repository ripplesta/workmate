# WorkMate(仮)
チャットbot機能付き進捗管理アプリ  

## 概要
- 自身の進捗管理や目標管理をでき、チャットbotなどでも管理、支援できるアプリ  
- Spring Boot/ Eclipse / Thymeleaf / PostgreSQLを使用

## 主機能
- ユーザー登録/ログイン機能  
- 進捗の一覧管理(完了/未完了など)・追加・更新・削除  
- チャット風インターフェースでの対話  

## 余裕があれば追加  
- カレンダー表示  
- 週報レポートの自動作成  
- メールなどの通知  
- タスクにソート機能(日付・時間やタスク名順など)
- 時間が見てわかりやすいように
